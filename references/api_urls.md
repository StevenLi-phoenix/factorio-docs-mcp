| Lua API（HTML） | https://lua-api.factorio.com/latest/ |
| Lua API（JSON） | https://lua-api.factorio.com/latest/runtime-api.json |
| 原型 API（JSON） | https://lua-api.factorio.com/latest/prototype-api.json |

| 原型文档 | https://lua-api.factorio.com/latest/prototypes.html |
| Factorio Wiki | https://wiki.factorio.com/ |
| Mod 门户 | https://mods.factorio.com/ |
| Mod 开发 Wiki | https://wiki.factorio.com/Modding |

## 按版本访问

替换 `latest` 为具体版本号，如 `2.0.28`：

```
https://lua-api.factorio.com/2.0.28/runtime-api.json
```
